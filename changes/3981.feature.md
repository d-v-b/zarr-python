Added `Array.lazy` accessor for deferred indexing. `arr.lazy[selection]` returns a
lazy view without performing I/O; call `.result()` (or `np.asarray()`) to
materialize. Composition (`view[a][b]`) and orthogonal/vectorized indexing
(`view.oindex[...]`, `view.vindex[...]`) all defer until materialization.
NumPy interop via `__array__` enables plug-and-play with matplotlib, dask,
and other array-consuming libraries.
